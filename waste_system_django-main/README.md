# 醫療廢棄物管理系統 (Medical Waste Management System)

這是一個基於 **Django 6.0 (Python)** 框架開發的醫療廢棄物管理系統，旨在協助醫院高效追蹤與管理廢棄物處理流程。
本專案由原本的 PHP 架構重構為 Django 架構，大幅提升了系統的安全性、可維護性與擴充性。

## 🚀 專案特色 (Features)

### 1. 視覺化儀表板 (Dashboard)
* **數據總覽**：整合 Chart.js 顯示回收物質、一般事業廢棄物、生物醫療廢棄物之產量與費用趨勢。
* **動態切換**：支援「近 12 個月」與「近 24 個月」的數據切換與放大檢視功能。
* **關鍵指標**：即時顯示各類廢棄物（如：紅袋、黃袋、洗腎桶）的重量統計與處理費用。

### 2. 廢棄物管理
* **多層級紀錄**：支援「院內單位」與「部門層級」的廢棄物重量申報與紀錄。
* **分類管理**：涵蓋感染性廢棄物（紅/黃袋）、非感染性廢棄物、回收物、藥用玻璃等詳細分類。

### 3. 聯單與報表系統
* **聯單追蹤**：完整的聯單管理介面（清除單、再利用單），掌握廢棄物去向。
* **自動化報表**：自動生成廢棄物月報表，支援篩選與匯出功能。

### 4. 權限管理系統 (RBAC)
* **角色權限控制**：內建使用者 (User)、角色 (Role)、權限 (Permission) 管理模組。
* **彈性配置**：管理員可針對不同職位（如：護理長、清潔人員、管理員）分配不同的系統操作權限。

## 🛠️ 技術棧 (Tech Stack)

* **後端框架**：Python 3.x, Django 6.0
* **前端技術**：HTML5, CSS3, JavaScript (原生), Chart.js (圖表庫), Lucide (圖示庫)
* **資料庫**：SQLite (開發環境預設) / 支援 PostgreSQL 或 MySQL
* **版本控制**：Git
